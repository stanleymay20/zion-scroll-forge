## Common Issues

### Issue: Cannot Log In

**Symptoms:**
- "Invalid credentials" error
- Login button not responding
- Redirected back to login page

**Solutions:**

1. **Check Credentials**
   - Verify email address is correct
   - Check for typos in password
   - Ensure Caps Lock is off

2. **Reset Password**
   - Click "Forgot Password"
   - Enter your email
   - Check inbox (and spam folder)
   - Follow reset link
   - Create new password

3. **Clear Browser Cache**
   ```
   Chrome: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
   Firefox: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
   Safari: Cmd+Option+E (Mac)
   ```

4. **Try Different Browser**
   - Test with Chrome, Firefox, or Safari
   - Disable browser extensions
   - Try incognito/private mode

5. **Check Account Status**
   - Account may be suspended
   - Email verification may be pending
   - Contact support if issue persists


### Issue: Page Not Loading

**Symptoms:**
- Blank white screen
- Infinite loading spinner
- "Something went wrong" error

**Solutions:**

1. **Check Internet Connection**
   - Verify you're connected to internet
   - Try loading other websites
   - Restart router if needed

2. **Refresh the Page**
   - Press F5 or Ctrl+R (Windows)
   - Press Cmd+R (Mac)
   - Hard refresh: Ctrl+Shift+R or Cmd+Shift+R

3. **Clear Browser Data**
   - Clear cache and cookies
   - Restart browser
   - Try again

4. **Check Service Status**
   - Visit status.scrolluniversity.org
   - Check for ongoing maintenance
   - Follow @ScrollUniversity on Twitter for updates

5. **Try Different Device**
   - Test on mobile device
   - Try different computer
   - Use mobile app if available

## Authentication Problems

### Issue: Email Verification Not Received

**Symptoms:**
- No verification email in inbox
- Cannot access account features
- "Please verify your email" message

**Solutions:**

1. **Check Spam Folder**
   - Look in spam/junk folder
   - Mark as "Not Spam" if found
   - Add noreply@scrolluniversity.org to contacts

2. **Resend Verification Email**
   - Go to login page
   - Click "Resend Verification Email"
   - Enter your email address
   - Check inbox again

3. **Check Email Address**
   - Verify correct email was used
   - Check for typos
   - Update email if incorrect

4. **Wait and Retry**
   - Email may take up to 10 minutes
   - Check again after waiting
   - Contact support if still not received

### Issue: Two-Factor Authentication Problems

**Symptoms:**
- Cannot receive 2FA code
- Code not working
- Locked out of account

**Solutions:**

1. **Check Time Sync**
   - Ensure device time is correct
   - Enable automatic time sync
   - Restart authenticator app

2. **Use Backup Codes**
   - Locate saved backup codes
   - Enter one backup code
   - Generate new backup codes

3. **Contact Support**
   - Email support@scrolluniversity.org
   - Provide account details
   - Verify identity
   - Support will disable 2FA temporarily

## Course Access Issues

### Issue: Cannot Access Enrolled Course

**Symptoms:**
- Course not showing in "My Courses"
- "Access Denied" error
- Course content not loading

**Solutions:**

1. **Verify Enrollment**
   - Go to "My Courses"
   - Check if course is listed
   - Verify payment was processed
   - Check email for enrollment confirmation

2. **Check Payment Status**
   - Navigate to "Billing History"
   - Verify payment completed
   - Check for failed transactions
   - Contact support if payment issue

3. **Clear Cache and Cookies**
   - Clear browser cache
   - Delete cookies
   - Restart browser
   - Try accessing again

4. **Check Course Status**
   - Course may be temporarily unavailable
   - Check for maintenance notifications
   - Contact instructor or support

### Issue: Video Lectures Not Playing

**Symptoms:**
- Black screen instead of video
- "Video unavailable" error
- Infinite buffering
- Audio but no video

**Solutions:**

1. **Check Internet Speed**
   - Test internet speed (minimum 5 Mbps)
   - Close other bandwidth-heavy applications
   - Try wired connection instead of WiFi
   - Contact ISP if speed is slow

2. **Browser Compatibility**
   - Update browser to latest version
   - Try different browser
   - Enable JavaScript
   - Disable ad blockers

3. **Video Quality Settings**
   - Lower video quality (720p or 480p)
   - Click settings icon in video player
   - Select lower resolution
   - Wait for buffering

4. **Clear Browser Cache**
   - Clear cache and cookies
   - Restart browser
   - Try again

## Payment Problems

### Issue: Payment Declined

**Symptoms:**
- "Payment failed" error
- Card declined message
- Transaction not completing

**Solutions:**

1. **Verify Card Details**
   - Check card number
   - Verify expiration date
   - Confirm CVV code
   - Check billing address

2. **Check Card Status**
   - Ensure card is active
   - Verify sufficient funds
   - Check for fraud holds
   - Contact your bank

3. **Try Different Payment Method**
   - Use different credit card
   - Try PayPal (if available)
   - Use ScrollCoin
   - Apply for scholarship

4. **Contact Support**
   - Email support@scrolluniversity.org
   - Provide transaction details
   - Include error message
   - Request manual enrollment

### Issue: ScrollCoin Transaction Failed

**Symptoms:**
- Transaction pending indefinitely
- "Insufficient balance" error
- Transaction not appearing in history

**Solutions:**

1. **Check Balance**
   - Navigate to ScrollCoin wallet
   - Verify sufficient balance
   - Account for transaction fees
   - Earn more ScrollCoin if needed

2. **Wait for Confirmation**
   - Blockchain transactions take time
   - Wait 5-10 minutes
   - Check transaction status
   - Refresh wallet page

3. **Check Network Status**
   - Verify Ethereum network status
   - Check for network congestion
   - Wait for network to clear
   - Try again later

4. **Contact Support**
   - Provide transaction hash
   - Include wallet address
   - Describe the issue
   - Request investigation

## Mobile App Issues

### Issue: App Won't Install

**Symptoms:**
- Installation fails
- "Cannot install" error
- App not appearing on home screen

**Solutions:**

1. **Check Device Compatibility**
   - iOS 13+ or Android 8+ required
   - Verify device meets requirements
   - Update operating system
   - Check available storage

2. **Clear Storage Space**
   - Delete unused apps
   - Clear cache
   - Remove old photos/videos
   - Free up at least 500MB

3. **Reinstall App**
   - Delete existing app
   - Restart device
   - Reinstall from browser
   - Follow installation prompts

### Issue: Offline Mode Not Working

**Symptoms:**
- Cannot access downloaded content
- "No internet connection" error
- Content not syncing

**Solutions:**

1. **Verify Downloads**
   - Check downloaded courses
   - Ensure downloads completed
   - Re-download if needed
   - Check storage space

2. **Enable Offline Mode**
   - Go to Settings
   - Enable "Offline Mode"
   - Download content while online
   - Test offline access

3. **Clear App Cache**
   - Go to app settings
   - Clear cache
   - Re-download content
   - Try again

## Error Messages

### "Session Expired"

**Cause:** Authentication token expired

**Solution:**
- Log out and log back in
- Enable "Remember Me"
- Check browser cookie settings

### "Access Denied"

**Cause:** Insufficient permissions

**Solution:**
- Verify your account role
- Check course enrollment
- Contact administrator
- Request access if needed

### "Server Error (500)"

**Cause:** Internal server error

**Solution:**
- Refresh the page
- Wait a few minutes
- Try again
- Contact support if persists

### "Not Found (404)"

**Cause:** Resource doesn't exist

**Solution:**
- Check URL for typos
- Verify resource exists
- Use search to find content
- Contact support
